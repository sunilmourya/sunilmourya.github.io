## What I worked on

Spent 2 hours on Rust iterators — specifically `flat_map`, `chain`, and writing custom `Iterator` impls.

Enforced the 30-minute rule. Sat with a broken `impl Iterator for MyStruct` without touching AI.
Turned out I was missing the `Item` associated type. Compiler error was pointing right at it.
That's 30 minutes I *won't* forget.

```rust
struct Counter {
    count: u32,
}

impl Iterator for Counter {
    type Item = u32;

    fn next(&mut self) -> Option<Self::Item> {
        if self.count < 5 {
            self.count += 1;
            Some(self.count)
        } else {
            None
        }
    }
}
```

The `type Item = u32` line. That's the thing I always let autocomplete fill in.
Not today.

## The Run

5.2 km. 5:48/km pace. First run of the 100 days.

Legs felt stiff in the first kilometer — classic. By km 3 the thinking started. Worked through
the iterator problem in my head without knowing it. Came home and fixed it in 10 minutes.

Running is debugging time.

## Honest reflection

The struggle was uncomfortable. The old instinct — tab to Copilot, accept suggestion, move on —
fired repeatedly. Ignored it.

The fix took longer. The memory is deeper.
