# Personal Tech & Running Blog

My personal blog built with Jekyll and GitHub Pages. Documenting my journey learning Rust and training for marathons.

## 🚀 Quick Start

1. Clone this repository
2. Install Jekyll: `gem install bundler jekyll`
3. Run locally: `jekyll serve`
4. Visit: `http://localhost:4000`

## 📝 Creating Posts

Create a new file in `_posts/` with format: `YYYY-MM-DD-title.md`
```markdown
---
layout: post
title: "Your Post Title"
date: 2024-11-22
category: Rust
tags: [rust, learning]
read_time: 5
excerpt: "Brief description..."
---

Your content here...
```
